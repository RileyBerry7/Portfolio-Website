College Map Itinerary Application (C++, Qt, SQL):

In a collaborative effort with fellow students, I contributed significantly to the development of a College Map Itinerary application using C++, Qt, and SQL. This project aimed to create a simulated map featuring numerous colleges across the United States. Users could compile a personalized list of colleges they intended to visit and then employ a range of common traversal algorithms within the application to navigate their map. My role encompassed extensive work on the GUI to ensure a user-friendly experience. Additionally, I implemented a parser that facilitated the conversion of data from a .csv file into our SQL database, streamlining the data management process. This project underscored my proficiency in C++, Qt, and SQL, as well as my ability to collaborate within a team to deliver a functional and efficient software solution.
